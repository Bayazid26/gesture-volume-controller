# ✋ Gesture Volume Controller (AI + Computer Vision)

## 🚀 Overview

Control your system volume using hand gestures in real time using OpenCV and MediaPipe.

---

## 🧠 Features

- Real-time hand tracking
- Gesture-based volume control
- Smooth audio transitions
- Noise filtering
- Clean modular Python design

---

## 🛠️ Tech Stack

- Python
- OpenCV
- MediaPipe
- Pycaw
- NumPy

---
